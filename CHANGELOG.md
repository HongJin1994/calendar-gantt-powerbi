# Changelog / 版本紀錄

## 1.0.0.0

繁體中文：

- 建立公開文件與支援 repository。
- 新增 Calendar Gantt 的使用說明、隱私權政策、支援方式與版本紀錄。
- 文件提供繁體中文與英文內容。
- 說明 Tooltips 可作為多個額外 hover 資訊欄位。
- 新增 Power BI 高對比模式支援。
- 一般模式的分類色彩改用 Power BI 色盤。
- 新增 Power BI rendering events 整合。
- 新增中英雙語 landing page 與空狀態提示。
- 新增 en-US 與 zh-TW 語系資源。
- 提高預設 WindowTime 透明度，並改善深色主題下分類色彩的可讀性。
- 新增甘特長條右鍵 Power BI context menu 支援。
- 新增跨 visual selection 支援。
- 新增鍵盤導覽與鍵盤選取支援。
- 新增 Power BI 官方 tooltip service 支援。

English:

- Initial public documentation and support repository.
- Documents Calendar Gantt usage, privacy, support, and release history.
- Documentation is available in Traditional Chinese and English.
- Documents Tooltips as additional hover information fields.
- Adds Power BI high contrast mode support.
- Uses the Power BI color palette for category colors in regular mode.
- Adds Power BI rendering events integration.
- Adds bilingual landing page and empty-state guidance.
- Adds en-US and zh-TW localization resources.
- Increases the default WindowTime opacity and improves category color readability on dark themes.
- Adds Power BI context menu support on schedule bars.
- Adds selection support across Power BI visuals.
- Adds keyboard navigation and keyboard selection support.
- Adds official Power BI tooltip service support.
