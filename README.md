# Calendar Gantt for Power BI

## 繁體中文

Calendar Gantt 是一個 Power BI 自訂視覺效果，用來將任務排程顯示成以週為單位的日曆式甘特圖。它適合用在需要比較任務、分類、標籤，以及最多三組時間區間的報表情境。

### 功能

- 以日曆週為基礎的甘特圖版面
- 每個任務最多支援三組開始/結束時間
- 支援顏色分類欄位與圖例
- 支援長條前後文字標籤
- 支援多個 Tooltips 額外資訊欄位
- 可調整標籤、長條高度、透明度、圓角與堆疊模式
- 一般模式使用 Power BI 色盤產生分類色彩
- 支援 Power BI 高對比模式
- 支援 Power BI rendering events 整合
- 支援中英雙語 landing page 與空狀態提示
- 支援 en-US 與 zh-TW 語系資源
- 支援長條右鍵 Power BI context menu
- 支援跨 visual selection
- 支援鍵盤導覽與鍵盤選取
- 支援 Power BI 官方 tooltip service

## English

Calendar Gantt is a Power BI custom visual for displaying task schedules across calendar weeks. It is designed for timelines where users need to compare tasks, categories, labels, and up to three time ranges in a compact calendar-based Gantt layout.

### Features

- Calendar-based Gantt layout for task schedules
- Supports up to three start/end time ranges per task
- Optional color category field for grouping and legend display
- Optional labels before and after each bar
- Optional Tooltips fields for additional hover information
- Formatting controls for labels, bar heights, opacity, radius, and stacking mode
- Uses the Power BI color palette for category colors in regular mode
- Supports Power BI high contrast mode
- Supports Power BI rendering events integration
- Includes bilingual landing page and empty-state guidance
- Includes en-US and zh-TW localization resources
- Supports Power BI context menu on schedule bars
- Supports selection across Power BI visuals
- Supports keyboard navigation and keyboard selection
- Supports the official Power BI tooltip service
