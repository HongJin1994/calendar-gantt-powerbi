# Support / 支援

## 繁體中文

請使用此 repository 回報 Calendar Gantt for Power BI 的問題或詢問使用方式。

### 開 issue 前請準備

請盡量提供：

- Power BI 環境：Desktop、Service、Report Server 或 Embedded
- Visual 版本
- 問題簡述
- 重現步驟
- 預期行為
- 實際行為
- 必要時可附上截圖

請不要提供機密商業資料、私人資料集、客戶資料、access token 或密碼。

### Issue Tracker

請在這裡開啟支援請求：

https://github.com/HongJin1994/calendar-gantt-powerbi/issues

### 功能建議

歡迎提出功能建議。請描述你的商業情境、期望行為，以及目前行為不足的地方。

## English

Use this repository to report issues or ask questions about Calendar Gantt for Power BI.

### Before Opening an Issue

Please include:

- Power BI environment: Desktop, Service, Report Server, or Embedded
- Visual version
- A short description of the problem
- Steps to reproduce the problem
- Expected behavior
- Actual behavior
- Screenshots if helpful

Do not include confidential business data, private datasets, customer data, access tokens, or passwords.

### Issue Tracker

Open a support request here:

https://github.com/HongJin1994/calendar-gantt-powerbi/issues

### Feature Requests

Feature requests are welcome. Please describe the business scenario, expected behavior, and why the current behavior is not enough.
