# AppSource Listing Draft / AppSource 上架草稿

## Product name / 產品名稱

Calendar Gantt

## Short description / 簡短描述

**繁中**  
用日曆方式呈現任務排程與多時段資料的 Power BI 自訂視覺。

**English**  
A Power BI custom visual that presents task schedules and multiple time ranges in a calendar-based Gantt layout.

## Long description / 詳細描述

**繁中**  
Calendar Gantt 是一個 Power BI 自訂視覺，適合用來顯示任務排程、跨天區間、不同時間段與分類資訊。它以月曆格狀方式呈現資料，讓使用者可以快速看出某項任務在哪幾天執行、不同時段如何重疊，以及各分類之間的分布狀態。  

此視覺支援格式面板設定、工具提示、選取、右鍵選單、鍵盤操作、高對比模式與多語系介面，適合用在排程追蹤、運作管理、服務排班與專案時程檢視等情境。

**English**  
Calendar Gantt is a Power BI custom visual designed for task schedules, date ranges, time windows, and category-based planning data. It uses a calendar-style grid to show when tasks happen, how time ranges overlap, and how categories are distributed across a month view.  

The visual supports formatting options, tooltips, selection, context menu actions, keyboard navigation, high contrast mode, and multilingual UI text. It is suitable for schedule tracking, operations planning, service dispatch, project timelines, and similar planning scenarios.

## Key features / 主要功能

- Calendar-based Gantt layout for monthly scheduling
- Multiple time ranges per task
- Category-based color mapping
- Tooltip fields for extra details
- Selection and cross-visual interaction
- Context menu support
- Keyboard navigation support
- High contrast support
- Bilingual UI strings (Traditional Chinese / English)
