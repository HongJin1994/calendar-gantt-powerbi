# Privacy Policy / 隱私權政策

Last updated / 最後更新：2026-05-18

## 繁體中文

Calendar Gantt for Power BI 設計為在 Microsoft Power BI 內顯示資料。

### 資料處理

此 visual 會呈現報表作者指派給 visual 的資料欄位。這些欄位可能包含任務名稱、分類值、標籤、tooltip 文字，以及日期或日期時間值。

### 外部網路存取

此 visual 在 Power BI capabilities 設定中不需要外部網路存取權限。它不是設計來將報表資料傳送到外部服務。

### 資料儲存

此 visual 不會刻意將報表資料儲存在 Power BI 外部。

### Microsoft Power BI

此 visual 是在 Power BI 中使用，因此也可能受到 Microsoft Power BI 服務條款、隱私權聲明、租用戶設定與管理員控制的規範。

### 聯絡方式

若有隱私或支援問題，請在此 repository 開 issue：

https://github.com/HongJin1994/calendar-gantt-powerbi/issues

## English

Calendar Gantt for Power BI is designed to display data inside Microsoft Power BI.

### Data Handling

The visual renders the data fields assigned to it by the report author. These fields may include task names, category values, labels, tooltip text, and date/time values.

### External Network Access

The visual does not require external web access privileges in its Power BI capabilities configuration. It is not designed to send report data to an external service.

### Data Storage

The visual does not intentionally store report data outside Power BI.

### Microsoft Power BI

Use of this visual occurs inside Power BI and may also be subject to Microsoft's Power BI terms, privacy statements, tenant settings, and administrator controls.

### Contact

For privacy or support questions, open an issue in this repository:
